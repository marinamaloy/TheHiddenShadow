# 透明人恐怖现象 视觉方案

## 1. 设计理念

### 整体视觉方向

整体视觉呈现冷峻、神秘且不安的氛围。美学上参考电影《降临》的极简科幻感与《鬼入侵》的哥特式恐怖，营造出一种"看不见的存在"所带来的心理压迫感。设计核心在于通过半透明、雾化和模糊效果，表现"透明人"的隐匿与无处不在，让观众感受到一种被无形之物注视的诡异体验。

### 内容密度

中等密度。每页聚焦一个核心恐怖现象，通过大面积留白和视觉元素（如雾气、模糊人影）营造氛围，而非堆砌信息。这种密度旨在引导观众沉浸于视觉叙事，强化恐怖的心理暗示。

## 2. 色彩系统

**主色调**

- 背景色：#0A0E17（深邃墨蓝，如深夜般压抑）
- 主色：#1A2536（青灰色，冷峻神秘）
- 辅助色：#2C3E50（深蓝灰，增添层次感）
- 强调色：#4A6D7C（青蓝色，用于光晕和特效）

**文字颜色**

- 深色文字（用于浅色背景）：#1A2536
- 浅色文字（用于深色背景）：#C0C5CE（冷灰白，如幽灵般飘渺）

## 3. 字体设计

**标题字体**

- 中文字体
  - Chiron Hei HK（现代几何黑体，线条锐利，科技感与冷峻感并存，契合"透明人"的科幻恐怖主题）
- 英文字体
  - Oranienbaum（高对比度衬线体，风格优雅而锐利，为标题增添一丝古典而诡异的气质）

**正文字体**

- 中文字体
  - Noto Sans SC（思源黑体，中性现代，可读性极佳，确保信息清晰传达，不干扰氛围营造）
- 英文字体
  - Quattrocento Sans（经典人文无衬线体，温和清晰，确保英文内容的可读性与专业感）

## 4. 图像风格

图像风格以冷色调、低饱和度的摄影为主，强调光影对比和模糊、半透明效果。多采用微距特写捕捉材质纹理（如雾气、玻璃、水面），并运用动态模糊或长曝光来表现"看不见的动作"。可叠加抽象的光影扭曲、数字噪点或微弱的冷光，营造出一种介于现实与超现实之间的诡异视觉体验。
